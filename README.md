# 📄 AI Resume Analyzer

AI Resume Analyzer is a Python-based web application that helps users evaluate their resumes by extracting text from PDF files, detecting technical skills, calculating a Resume Strength Score, and recommending suitable job roles.

## 🚀 Features

* Upload Resume in PDF Format
* Automatic Resume Text Extraction
* Technical Skill Detection
* Resume Strength Score Calculation
* Detailed Score Breakdown
* Recommended Job Roles
* Missing Skills Suggestions
* Resume Category Classification
* Interactive Web Interface using Streamlit

## 🛠️ Technologies Used

* Python
* Streamlit
* PDFPlumber

## 📊 Resume Evaluation Parameters

The application evaluates resumes based on:

* Technical Skills
* Projects
* Certifications
* Education
* GitHub Profile
* LinkedIn Profile

## 💼 Job Roles Recommended

Depending on the detected skills, the application can recommend roles such as:

* Software Engineer
* Frontend Developer
* Data Analyst
* AI / ML Engineer
* DevOps Engineer
* Cybersecurity Analyst

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ai-resume-analyzer.git
cd ai-resume-analyzer
```

Install dependencies:

```bash
pip install streamlit pdfplumber
```

Run the application:

```bash
streamlit run app.py
```
