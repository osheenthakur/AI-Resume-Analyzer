# AI Resume Analyzer

An AI-powered resume analysis tool built using Python and Streamlit.

## Features

* Upload PDF resumes
* Extract resume text automatically
* Detect technical skills
* Generate a resume score
* Identify strengths
* Recommend missing skills
* Provide resume improvement suggestions

## Technologies Used

* Python
* Streamlit
* PDFPlumber
* GitHub

## Future Improvements

* Gemini AI integration
* ATS score analysis
* Job role recommendations
* Resume keyword optimization
* Resume comparison with job descriptions
